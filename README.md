# TP3 – Modo Protegido en x86
https://github.com/cirosantilli/x86-bare-metal-examples/tree/528ab2becc4992218ca0b7e297f3e1f1674268de

## Descripción

Este repositorio contiene el desarrollo del Trabajo Práctico N.º 3, dedicado al estudio del **modo protegido** en procesadores x86.

A lo largo de este trabajo se analiza cómo un procesador x86 evoluciona desde su estado inicial, conocido como **modo real**, hasta alcanzar el **modo protegido**, una etapa que le permite acceder a funciones más avanzadas, mayor seguridad y un mejor aprovechamiento de sus recursos.

Además del análisis teórico, se ejecuta un ejemplo práctico en una máquina virtual para observar paso a paso este proceso de transición y comprender qué ocurre internamente en el procesador.

## Objetivos

* Comprender cómo inicia un procesador x86.
* Diferenciar el modo real del modo protegido.
